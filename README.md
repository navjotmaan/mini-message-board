# mini-message-board

A lightweight, server-side rendered message board application built with Node.js and Express, featuring dynamic content delivery via EJS templates and in-memory data management.

### Tech-stack
- **Runtime environment:** Node.js
- **Framework:** Express.js
- **Template engine:** EJS

### Features
- **Server-Side Rendering:** Utilizing EJS to dynamically generate HTML templates on the server.
- **New Message Submission:** A "New Message" form that allows users to add message to the board in real-time.
- **Message Details:** Users can click on any message to view the message details.

### Getting started
```
npm install
node app.js
```
